Ideas

As we have been indicating throughout this book, JavaScript has become more and more powerful.  Also p5.js has a few interesting capabilities worth while exploring.  In this final section we want to explore a couple of those, such as

  • web workers
  • playing videos
  • accessing the webcam
  • accessing the microphone
  • web workers
  • web sockets
  • 3D drawing

In addition, access an online database like Back4App might open up a whole new set of applications.  And an mobile devices accessing the sensors would allow for another set of applications.  And finally, peer-to-peer would be a cool feature.
